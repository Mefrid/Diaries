  Diaries application
</h2>

## Description

The application provides the following features:

- Create diaries
- Change the name of the diary
- Delete diaries
- Create diary notes
- Change the title and content of the note in the diary
- Delete notes

Frontend technologies:

- React
- Material-UI
- Formik
- Yup

Backend technologies:

- NestJS

## Quick start

### Frontend

```sh
cd frontend
yarn
yarn start
```

### Backend

```sh
cd backend
npm install
npm run start:dev
```

## Build whole application

```sh
npm run build
```
